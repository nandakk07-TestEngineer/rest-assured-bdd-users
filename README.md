# 🧪 REST Assured BDD Automation Framework

This project is a Behavior-Driven Development (BDD) API automation framework built using **Rest Assured**, **Cucumber**, and **Java**. It is designed to validate user-related APIs with clean structure, reusability, and maintainability.

---

## 🚀 Tech Stack

- Java
- Rest Assured
- Cucumber (BDD)
- TestNG / JUnit
- Maven
- JSON
